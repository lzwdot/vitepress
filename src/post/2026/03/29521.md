# Windows/WSL 环境下安装 OpenClaw 并接入 DeepSeek

## 环境准备

- **操作系统**: Windows + WSL (Ubuntu 22.04.5 LTS)
- **Node.js 版本**: v22.20.0
- **DeepSeek**：API Key

## 安装 OpenClaw

### 安装命令

根据不同环境选择对应的安装脚本：

**Windows 环境：**

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

**WSL 环境：**

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### 配置引导

安装完成后会自动进入 `OpenClaw onboarding` 引导流程，按照提示进行配置：

## 接入 DeepSeek

### 1. 获取 API Key

首先需要前往 DeepSeek 开放平台申请 API Key：

访问地址：https://platform.deepseek.com/api_keys

![获取 API Key](./images/29521-02.png)

### 2. 配置 OpenClaw 模型

在 OpenClaw 控制台中配置 DeepSeek 模型：

```json
{
  "models": {
    "providers": {
      "deepseek": {
        "baseUrl": "https://api.deepseek.com/v1",
        "apiKey": "你的 DeepSeek API key",
        "api": "openai-completions",
        "models": [
          {
            "id": "deepseek-chat",
            "name": "DeepSeek Chat"
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "deepseek/deepseek-chat"
      }
    }
  }
}
```

**配置说明：**

- `baseUrl`: DeepSeek API 的基础 URL
- `apiKey`: 替换为你申请的 DeepSeek API key
- `api`: 使用 OpenAI 兼容的 API 格式
- `models`: 配置可用的模型列表
- `agents.defaults.model.primary`: 设置默认使用的模型

### 3. 保存并重启配置

点击 **Update** 按钮保存配置并重启服务。

![更新配置](./images/29521-03.png)
